# 📸 Instagram Clone

A full-stack Instagram-like social media application built using modern web technologies. This project includes core social features like authentication, posting, messaging, and profile management.

---

## 🚀 Features

* 🔐 User Authentication (Login & Register)
* 📝 Create, Edit, and Delete Posts
* ❤️ Like and Interact with Posts
* 👤 Edit User Profile
* 💬 One-to-One Real-time Messaging using Socket.IO
* 📨 Chat system with live updates
* 🔓 Logout functionality

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* JavaScript
* Tailwind CSS

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB

### Real-time Communication:

* Socket.IO
